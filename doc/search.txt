Advertisement document fields:
------------------------------
* Category (XC: select from existing list (including "digital product or service), add new category)
* Title (S: text)
* Description (XD: text)
* Price (number)
* Condition (list: new, other(see details), seller refurbished, used, for parts or not working)
* Shipping (list: local pickup, delivery, mail, TBD)
* Buying format (list: accepts offers, auction, buy it now)
* Item location (list: city, state, country, continent) (N: ISO country code)

* Keyword (K:)
* Author (A:)
